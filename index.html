HTML & CSS CheatSheet

Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The wrapper element for all HTML tables.
<tr> Table Row The table row container.
<td> Table Data The table data container.

HTML Tags

Element Name Description
<html> HTML It is the root element and it defines the whole HTML document.
<body> Body It defines the document's body.
<p> Paragraph It defines a paragraph.
<div> Division Defines a section in a document
<h1> to <h6> Heading Defines a heading in a document
